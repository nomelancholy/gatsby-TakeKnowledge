### 기본행연산

m x n 행렬 A에 관한 세가지 기본행연산은 다음과 같다(단, 1<= i, j <= m 이며 i != j)

1. R<sub>i,j</sub> : A의 i번째 행과 j번째 행을 교환한다.
2. R<sub>i</sub>(c) : A의 i번째 행에 0이 아닌 상수 c를 곱한다
3. R<sub>i,j</sub>(c) : A의 i번째 행에 상수 c를 곱하여 j번째 행에 더한다.

영행 : 원소기 모두 0인 행

#### 행제형의 정의

1. 영행이 있다면 그것은 영행이 아닌 행의 아래에 있다.
2. 영행이 아닌 행의 첫번째 0이 아닌 원소를 그 행의 선도원소라고 할 때 모든 선도원소는 1이다.
3. 영행이 아닌 연속된 두행을 i번째 행과 i + 1 번째 행이라 할 때 i번째 행의 선도원소는 i + 1 번째 행의 선도원소보다 왼쪽에 있다(i >= 1)

#### 소거행제형의 정의

1. i번째 행의 선도원소가 j번째 열에 있다면 j번째 열의 다른 모든 원소는 0이다.

## 행렬연산

### 기본 개념

#### 정방행렬과 주대각 원소

A를 m x n 행렬이라고 할 때 A의 i번째 행의 j번째 원소를 A의 (i,j) 원소라 부르며 A = (a<sub>ij</sub>)로 표시한다 (1 <= i <= m, 1 <= j <= n). 이 때 m = n이면 A를 차수가 n인 정방행렬 또는 n차 정방행렬이라하고 이때 A의 모든 (i,i) 원소를 A의 주대각 원소라고 한다.

#### 대각행렬, 삼각행렬, 스칼라 행렬, 단위행렬

A = (a<sub>ij</sub>) 를 n차 정방행렬이라고 하자. i != j 를 만족하는 모든 i, j에 대해 a<sub>ij</sub> = 0일 때 A를 대각행렬이라 한다. 또한 i < j를 만족하는 모든 i,j 에 대해 a<sub>ij</sub> = 0 일 때 하삼각행렬이라 하고 i > j를 만족하는 모든 i,j 에 대해 a<sub>ij</sub> = 0 일 때 상삼각행렬이라 하며 이 두 종류의 행렬을 묶어서 삼각행렬이라 한다. 정방행렬 중 주대각원소의 값이 동일한 행렬은 스칼라 행렬이라 하고 정방행렬 중 주대각 원소가 모두 1인 대각행렬을 단위행렬이라 한다.

### 행렬의 전치

주어진 행렬의 행과 열을 서로 바꾸어 배치하는 것을 행렬의 전치라 한다. A = (a<sub>ij</sub>) 를 m x n 행렬이라 하면 A의 전치 행렬은 m x n 행렬 A<sup>T</sup> = ((a<sup>T</sup>)<sub>ij</sub>)로 다음과 같이 정의한다.

(a<sup>T</sup>)<sub>ij</sub> = a<sub>ji</sub>

A<sup>T</sup> = A인 행렬은 대칭행렬이라 한다.

### 정직행렬과 역행렬

n차 정방행렬 A에 대해 행렬 B가 존재하여

AB = BA = I<sub>n</sub>

을 만족할 때 A를 정칙행렬이라하고 B를 A의 역행렬이라 하며 B = A<sup>-1</sup>로 나타낼 수 있습니다.

## 행렬식

### 행렬식

행렬식이란 정방행렬에 실수를 대응시키는 함수이다. 즉 하나의 행렬이 주어지면 그 행렬에 고유한 실수값 하나를 할당할 수 있다. 정행행렬 A의 행렬식은 | A | 또는 det A로 나타낸다.

A= (a<sub>ij</sub>)를 n차 정방행렬이라 할 때 A에서 i번째 행과 j번째 열을 제거시켜 구성되는 (n-1)차 정방행렬을 A의 (i,j) 소행렬이라하고 그 소행렬의 행렬식을 A의 원소 a<sub>ij</sub>에 대한 소행렬식이라 하고 M<sub>ij</sub> 로 표시한다. 또한

A<sub>ij</sub> = (-1)<sup>i+j</sup>M<sub>ij</sub>

로 정의되는 A<sub>ij</sub>를 A의 원소 a<sub>ij</sub>에 대한 여인수라고 한다.

## 평면벡터와 공간벡터

### 평면벡터

A를 평면 R<sup>2</sup> 상의 벡터라고 하면 벡터 A의 시작점을 평면의 원점에 맞출 때 끝점 P를 (a,b)라 하며 백터 A를 다음과 같이 정의할 수 있습니다.

A = <b>OP</b> = (a,b)

이때 a를 벡터 A의 x축 석ㅇ분이라고 하고 b를 벡터 A의 y축 성분이라고 합니다. 특별히 시작점과 끝점이 모두 평면의 원점일 때의 벡터는 영벡터라 하며 O = (0,0) 으로 표시합니다. 또 벡터 A=(a,b)와 벡터 B=(c,d)가 같다고 하는 것은 끝점이 같아야 하므로 a=c이고 b=d임을 뜻합니다.

평면 벡터 A=(a,b) 가 주어졌을 때 벡터 A의 크기를 |A|라 쓰고

|A| = √(a^2 + b^2) 과 같이 정의합니다. 즉 벡터 A의 크기 |A|는 원점에서 점(a,b) 까지 거리를 뜻합니다.

### R^3 공간벡터

A를 평면 3차원 공간 R<sup>3</sup> 상의 벡터라고 하면 벡터 A의 시작점을 평면의 원점에 맞출 때 끝점 P를 (a,b,c)라 하면 백터 A를 다음과 같이 정의할 수 있습니다.

A = (a,b,c)

이때 a를 x축 성분, b를 y축 성분, c를 z축 성분이라고 합니다.

영벡터, 벡터 크기, 실수곱, 벡터합은 2차원과 같습니다.

### R^n 공간벡터

### 벡터의 내적

R<sup>n</sup> 공간의 두 백터를 A = (a<sub>1</sub>, a<sub>2</sub>, ... a<sub>n</sub>), B = (b<sub>1</sub>, b<sub>2</sub>, ... b<sub>n</sub>) 이라 할 때, A와 벡터 B의 내적은 다음과 같이 정의합니다.

A · B = a<sub>1</sub>b<sub>1</sub> + a<sub>2</sub>b<sub>2</sub> + ... + a<sub>n</sub>b<sub>n</sub>
